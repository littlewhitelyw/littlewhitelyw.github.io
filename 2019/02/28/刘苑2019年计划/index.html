刘苑的2019年心愿

发表于 2019-02-28

朋友圈翻到同事晒的年假照片，带着女儿去东南亚玩滑翔伞了。

然后想到，这也是我一直想体验的吖~~ 💓

所以脑子里突然冒出一个 想法 💡

不如把一直以来想做没做打算做的事都列出来
作为2019的小心愿吧
看看今年剩下的十个月能幸运地实现几件呢 🤗🤗🤗

  • 体验滑翔伞（哈哈哈，以这个开头吧）
  • 学潜水
  • 香港、日本、韩国、东南亚任选一个城市旅行，总之要温暖的季节去
  • 学滑板
  • 坚持弹三个星期的哈农，不弹流行曲子，每天练至少一个小时。
  • 读20本书
  • 驾照拿到手
  • 把我和李的网站内容丰富起来（写本文的动力之一）
  • 买个轻薄笔记本
  • 考个英语的证书？
  • 王者上分？哈哈哈
  • 2019年12月31号体重不超105
  • 暂时想这么多，我还会添进来的，哈哈哈
